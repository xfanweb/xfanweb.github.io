前言

2.以下许可证，选择其一即可

3.安装完成，哎就很棒。

1.此工具安装比较简单，我就不一步一步来了，疯狂下一步就完了。

2.下一步后安装完成。

3.下面进行破解（破解时必须断网）！！！退出所有杀毒软件

双击“Navicat Keygen Patch v5.6.0 DFoX”文件

点击 Patch，找到刚刚安装的路径，找到“Navicat.exe”，如下图

出现如下图即为成功

2、3 如下图选择

在 4 中点击 generate，会在 Serial Keygen 内生成一串码

回到 Navicat，出现提示选择注册

粘贴秘钥，然后点击激活按钮

在弹出的界面选择手动激活 ，将请求码粘贴到注册机 Request Code 框中（完整过程看图）

4.激活成功

PyCharm

1.安装 PyCharm

1.打开安装包，点击 Next

2.弹框会提示选择安装路径，可以选择默认路径，我这里选择的是 D 盘，路径确定好后点击 Next

3.勾选 Create Desktop Shortcut 创建桌面快捷方式，再点击 Next

4.点击 Install 按钮，开始安装

5.等待安装完成后，勾选 Run IntellJ PyCharm, 点击 Finish 运行 PyCharm

6.PyCharm 运行成功后，点击 Exit 退出

2.下载补丁

将补丁 /jetbra 复制电脑某个位置，我放在了 E:/ 盘根目录下

补丁所属文件夹需单独存放，且放置的路径不要有中文与空格，以免 PyCharm 读取补丁错误。

3.开始破解

点击进入 /jetbra 补丁目录，再点击进入 /scripts 文件夹，双击执行 install-current-user.vbs 破解脚本

如果执行脚本被安全软件提示有风险拦截，允许执行即可。

提示安装补丁需要等待数秒。点击【确定】按钮后，过程大概 10 - 30 秒，如看到弹框提示 Done 时，表示激活成功

4.重启 PyCharm

5.输入指定激活码完成激活

粘贴到输入框内，点击 Activate 按钮，就激活成功了。

如果遇到重启 PyCharm 填入激活码依然无法激活，重启一下系统
